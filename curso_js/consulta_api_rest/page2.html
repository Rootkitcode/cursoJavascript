Que es Fetch?

Es el nombre de una nueva API para javascript con la cual podremos realizar
peticiones de tipo HTTP asincronas utilizando promesas haciendo que de alguna manera el codigo sea mucho mas sencillo.


Promise o promesa en javascript es un objeto que representa la terminacion o el frcaso de una operacion asincrona.
Dado que la mayoria de las personas consumen promises ya creadas, en resumen la promesa o promise es un objeto que representa una eventual finalizacion o fallo.


En javascrip la asincronia es uno de los pilares fundamentales, ya que es un lenguaje de programacion de un solo subproceso o hilo(single thread) lo que significa que solo puede ejecutar una cosa a la vez.


En Javascript existen dos palabras reservadas Async y await, Async es una funcion que define asincronia, la cual devuelve
un objeto AsyncFunction, y es posible definir funciones asincronas a travez de la funcion async.


La expresion await provoca que la ejecucion de una funcion async sea pausada hasta que una promise sea terminada o rechazada y regresa a la ejecucion de la funcion async despues del termino.
